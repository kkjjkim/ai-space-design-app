# PRD — 사업기획 + 공간브랜딩 상담 신청 사이트

> 사람이 보는 기획서. 코딩 도구에는 이 파일과 `AGENTS.md`를 함께 넣는다.
> `[ ]` 는 대표님이 채울 자리. AI는 비워두고 지어내지 않는다.

---

## 한 줄 요약
- 이 사이트는 **고급 매장·레스토랑·대형카페를 준비하는 사장**을 위해, **국가공인 경영지도사 + 디자인드비**가 **창업 첫걸음부터 매출까지 한 번에 도와주는 곳**임을 보여주고 **무료 상담을 신청**하게 만드는 사이트다.

## 목적
- 인테리어를 "사업·브랜드 기획부터" 함께하는 차별화된 팀임을 알리고, 양질의 상담 신청(고객 DB)을 모은다.
- 모인 신청은 운영자가 검토해 디자인드비(주력) 또는 다른 시공 파트너로 연결한다. (이 분배는 사이트 밖에서 운영자가 수동으로 한다.)

## 타깃 (한 명을 구체적으로)
- 누구: 고급 레스토랑 / 대형 카페·베이커리 / 내 브랜드 매장을 준비하는 사장. 예비 창업자 포함. "싸게"보다 "제대로, 브랜드답게"를 원하고 돈 쓸 의향이 있다.
- 그 사람의 고민: "업체는 많은데 내 사업과 브랜드를 이해하는 곳이 없다. 큰돈 들여 공사했는데 평범한 가게가 될까 봐 두렵다. 견적만 싸게 부르는 곳 말고, 장사 잘 되는 매장을 같이 기획해줄 곳을 원한다."

## 핵심 행동 (딱 하나)
- **무료 컨셉 상담 신청** (신청 폼 제출). 사이트의 모든 것은 이 행동 하나를 위해 존재한다.

## 성공 기준
- 한 달에 상담 신청(DB) **30건 이상**.
- 상담 → 계약 전환 약 20% (월 5~6건 계약 목표).

## 화면 목록
- **메인** (원페이지 스크롤. 이 한 장만으로 설득 + 신청이 끝나야 함)
- **포트폴리오** (시공 사례)
- **회사·신뢰** (자격·실적·팀)
- **서비스** (경영지도사 × 디자인드비 협업 방식)
- **신청 완료** (폼 제출 후)
- 공통: 상단 내비게이션 + 스크롤 따라다니는 고정 "무료 상담" 버튼

## 화면별 내용

### 메인 (위에서 아래 순서. 카피는 이 문구 그대로 사용)

**1) 히어로** (풀블리드 대형 이미지 또는 배경영상 — 운영자가 넣음)
- 큰 제목: **인테리어가 아니라, 장사 되는 브랜드를 짓습니다.**
- 부제: 국가공인 경영지도사의 사업 기획 + 백화점·명품 매장을 시공한 디자인드비. 창업 첫걸음부터 매출까지, 한 곳에서.
- 버튼: **무료 컨셉 상담받기 · 1분**
- 버튼 밑 작게: 영업 전화 안 합니다. 방향부터 들어드려요.

**2) 공감**
- 제목: 큰 돈 들어가는데, 평범한 가게가 될까 봐 두렵지 않으세요?
- 본문: 가게 하나 차리는 데 적은 돈이 들지 않죠. 막상 열고 보면 "어디서 본 듯한 가게"가 됩니다. 예쁘게는 했는데 손님이 한 번 오고 다시 안 와요. 업체 말은 다 비슷하고요.

**3) 진짜 문제**
- 제목: 망하는 진짜 이유는, 컨셉 없이 공사부터 시작해서입니다.
- 본문: 예쁜 인테리어는 기본이에요. "여긴 왜 와야 하지?"가 없으면 돈을 써도 그냥 또 하나의 가게예요. 그래서 공사보다 사업·브랜드부터 잡습니다.

**4) 왜 우리가 다른가** (2열 비교표 — 좌: 일반 인테리어 업체 / 우: 우리)
- 목적: 예쁘게 꾸미기 / 매출 나는 브랜드 만들기
- 방식: 기획·설계·시공 따로따로 / 사업 기획부터 시공·운영까지 한 팀
- 근거: 디자이너 감각에 의존 / 고객 동선·체류시간 데이터 기반(특허 출원)
- 사후: 시공 끝나면 끝 / 오픈 후 매출 데이터로 개선

**5) 우리 방식** (4단계 카드)
- 제목: 사업의 첫걸음부터 매출까지, 이렇게 함께합니다.
- 1. 사업·브랜드 방향 잡기 — 국가공인 경영지도사가 "되는 장사인지"부터.
- 2. 철학이 보이는 공간 설계 — 들어선 순간 "여긴 다르다"가 느껴지게.
- 3. 검증된 프리미엄 시공 — 백화점·명품 매장을 시공한 팀이 그대로 구현.
- 4. 오픈 후 매출까지 — 동선·체류시간 데이터로 계속 개선.
- 사례 한 줄(작게): 관광단지 안 대형 카페를 '도산서원' 컨셉으로 — 한옥의 결을 살린 공간에 한국식 빵과 차. 공간·메뉴·이름이 하나의 이야기로 묶이자 일부러 찾아오는 곳이 됐습니다.

**6) 증거 (신뢰)**
- 제목: 대형 백화점과 글로벌 명품 매장을 만들던 팀입니다.
- 본문: 까다로운 대형 유통사와 명품 브랜드가 믿고 맡긴 실력을, 이제 당신의 매장에 씁니다.
- 배지 3개: 공간 최적화 특허 출원 / 정부 인정 R&D 전담부서 / 실내건축공사업 면허 보유
- 숫자: 완공 [ ]곳 · 업계 경력 [ ]년 · [성과 한 줄]
- 포트폴리오 대형 사진 슬라이드(좌우 화살표 = 둥근 외곽선 버튼 + "01 / 08" 카운터). 사진은 운영자가 넣음(로고 없는 컷).
- 버튼: 포트폴리오 더 보기 → 포트폴리오 페이지

**7) 가격 뒤집기**
- 제목: 컨설팅은 '추가 비용'이 아니라, 망하지 않으려는 첫 단계입니다.
- 본문: 정말 비싼 건, 큰 돈 들여 만든 평범한 가게예요. 방향을 먼저 잡으면 헛돈을 줄이고 더 오래갑니다.

**8) 신청 (폼)**
- 제목: 머릿속 구상만 들고 오세요.
- 본문: 아직 정리 안 되셨어도 괜찮습니다. "이런 가게 하고 싶다" 한마디면 시작이에요. 무료고, 1분이면 됩니다.
- 폼 항목: 이름 / 연락처 / 업종 / 매장 위치·평수 / 예산대(선택, 드롭다운) / 하고 싶은 가게 한 줄
- 버튼: 무료 컨셉 상담받기
- 버튼 밑 작게: 영업 전화 없음 · 상담 후 결정은 자유

### 포트폴리오
- 제목: 우리가 만든 공간
- 한 줄: 대형 백화점과 글로벌 명품 매장을 시공한 실력입니다.
- 프로젝트 카드 그리드 + 대형 사진 슬라이드. 업종 필터(리테일 / F&B / 명품·뷰티). 사진·내용은 운영자가 넣음.
- 브랜드명 직접 노출은 공개 가능 여부 확인 전까지 "글로벌 명품 매장", "대형 백화점"으로 표기.
- 페이지 끝에 신청 버튼.

### 회사·신뢰
- 제목: 왜 믿을 수 있나요?
- 철학 한 단락: "공간 너머 브랜드 가치를 디자인한다."
- 신뢰 카드: 공간 최적화 시스템 특허 출원 / 정부 인정 R&D 전담부서 / 실내건축공사업 면허 / 사업자·납세 투명성.
- 대표·팀: 디자인 전공 출신, 업계 경력 [ ]년 (상세는 운영자가 채움).
- 끝에 신청 버튼.

### 서비스 (협업 방식)
- 제목: 사업의 뇌 + 공간의 뇌
- 두 칸: (좌) 국가공인 경영지도사 — 창업·사업 기획·자금·경영 전략 / (우) 디자인드비 — 공간 브랜딩·프리미엄 시공·데이터 운영.
- 합쳐지는 한 줄: "창업 첫걸음부터 매출까지, 끊김 없이."
- 끝에 신청 버튼.

### 신청 완료
- 신청 완료됐습니다. 1~2일 안에 연락드릴게요. 그동안 머릿속 구상을 편하게 정리해 두세요.

## 데이터 (저장)
- 저장할 정보(상담 신청 = leads 테이블):
  - 이름, 연락처, 업종, 매장 위치·평수, 예산대(선택), 하고 싶은 가게 한 줄(메모), 생성일시.
  - 유입 참고용(선택): 유입 경로/UTM.
- 신청이 들어오면 운영자에게 **알림 메일** 발송.
- 신청 목록 확인은 v1에서는 **Supabase 테이블에서 직접 확인**(별도 관리자 화면은 이번 범위 밖).

## 연동
- 폼/저장: Supabase (leads 테이블에 저장).
- 메일/알림: 신청 시 운영자에게 알림 (Resend 등 메일 발송 — 키는 환경변수).
- 결제: 없음.
- 로그인: 고객용 로그인 없음.

## 안 할 것 (이번 범위 밖)
- "비교견적" 류 기능·문구. (절대 금지)
- 결제, 고객 회원가입/로그인.
- 복잡한 관리자 대시보드 (v1은 Supabase 테이블로 신청 확인).
- 블로그/CMS, 다국어, 실시간 채팅.
- AI로 만든 가짜 인테리어를 '실제 시공 사례'처럼 쓰는 것. (히어로의 분위기 이미지/영상은 가능, 포트폴리오는 실사진만)

## 기술
- 스택: 풀 웹앱 (`02_표준스택` 따름) — Next.js(App Router) + Supabase + Vercel + Tailwind + shadcn/ui + TypeScript.
- 이유: 신청 데이터를 저장하고 다시 꺼내 봐야 하므로 가벼운 랜딩이 아니라 풀 웹앱.

## 디자인 방향 (AvroKO 톤)
- 분위기: 고급 에디토리얼·갤러리. 절제·여백·큰 사진. 차분하고 비싸 보이게.
- 배경: 따뜻한 아이보리/크림(순백 아님). 글자: 짙은 차콜. 포인트: 골드/브론즈 1색만, 아주 절제.
- 폰트: 큰 제목·인용문 = 명조(세리프, 예: Noto Serif KR / 본명조). 메뉴·본문·버튼·작은 글씨 = Pretendard(산세리프). 본문 16px 이상.
- 사진은 풀블리드 대형. 한 화면에 한 가지만 강조. 여백 넉넉.
- 모바일 우선. 무거운 효과로 느려지지 않게. 스크롤 시 은은한 페이드인 정도만.
- 단, 우리는 상담을 받아야 하므로 "무료 상담" 버튼은 또렷하게(골드 또는 차콜 채움) + 스크롤 고정 버튼. AvroKO식 약한 텍스트 링크로 숨기지 말 것.
- 참고 사이트: avroko.com (톤·여백·타이포), zorge9.estate (시네마틱 스크롤 — 단 가볍게).
- 브랜드명/로고: 아직 미정 → `[브랜드명]` 으로 두고, 정해지면 교체.
